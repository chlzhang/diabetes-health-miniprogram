# 健康生活助手小程序

一款面向关注健康的家庭与个人的微信小程序，提供**饮食记录 + 运动记录 + 数值打卡 + 家庭互助**四大核心功能，通过每日打卡和家人监督帮助建立健康生活习惯。

> 本应用是健康数据记录与生活辅助工具，所有内容仅供参考，不构成医疗建议。如有健康相关问题，请咨询专业医生。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 微信小程序原生开发（WXML + WXSS + JS） |
| 后端 | 微信云开发（CloudBase） |
| 数据库 | 云开发数据库（MongoDB-like 文档数据库） |
| 部署 | 云函数（Node.js 运行时） |

| 集合 | 说明 |
|------|------|
| `users` | 用户信息（openid、昵称、头像、身高体重、家庭组ID） |
| `families` | 家庭组（名称、6位邀请码、有效期、创建者、成员列表） |
| `meals` | 饮食记录（餐次、主食信息、营养选项、健康评级） |

| `blood_sugars` | 健康指标记录（时段、数值、状态） |
| `likes` | 点赞记录（点赞者、目标类型、目标ID） |

## 核心功能

### 首页看板
- 今日三餐记录状态 + 健康评级（健康/一般/注意）
- 本周运动时长进度条（目标：每周 ≥ 150 分钟中等强度运动）
- 最近健康指标数值 + 状态标记（正常/偏高/偏低）
- 快速记录入口（记饮食 / 记运动 / 记指标）
- 家庭成员今日打卡状态

### 饮食记录
- 支持早餐/午餐/晚餐/加餐四种餐次
- 记录主食类型和份量、蔬菜/蛋白质/甜食/含糖饮料选项
- 自动健康评级：`good`（≤150g 主食 + 蔬菜 + 蛋白 + 无甜食）/ `ok`（≤200g + 蔬菜或蛋白）/ `warn`（其他）

### 运动记录
- 8种预设运动类型（快走、慢跑、骑车、游泳、瑜伽、力量训练、太极、广场舞）
- 基于 MET 公式自动估算热量消耗：`MET × 体重(kg) × 时长(h)`
- 每周运动进度追踪

### 健康指标打卡
- 支持空腹/餐后2h/睡前三个时段
- 自动判定数值状态：正常 / 偏高（≥参考值）/ 偏低（<3.9 mmol/L）
- 异常值提示
- 数值范围基于常见健康人群参考标准，**不构成医疗诊断**

### 家庭互助
- 创建家庭组，生成6位邀请码（有效期7天）
- 加入家庭组（输入邀请码，上限8人）
- 家庭看板展示每位成员今日打卡状态
- 点赞互动

## 快速开始

### 环境要求
- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 最新稳定版
- 小程序基础库 ≥ 2.2.3（需支持云能力）
- 微信云开发环境

### 配置步骤

1. 克隆项目到本地
2. 用微信开发者工具打开项目根目录
3. 在 `miniprogram/app.js` 中将 `env: 'your-env-id'` 替换为你的云开发环境 ID
4. 在开发者工具中开通云开发，创建以下数据库集合：
   - `users`、`families`、`meals`、`exercises`、`blood_sugars`、`likes`
5. 右键点击 `cloud/functions/` 下的每个云函数文件夹，选择"上传并部署：云端安装依赖"
6. 编译预览即可运行

### 云函数列表

每个云函数目录包含 `index.js`（入口逻辑）和 `package.json`（依赖声明），使用 `wx-server-sdk` 连接云开发数据库。
